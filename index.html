Avatar

Avatar is used to represent a user or Profile picture. Here we have created different avatar can be used in website.

Default Avatar

As a Default Avatar is our square profile picture. We can keep it as it is or use styling with it.
We used class as avatar-image

Circular Avatar

As a Circular Avatar we are making avatar in round shape. We are using class as avatar-image and round-image. Usinground-image class image's border radius is set to 50% because of it changes into circular.

Avatar with border

As Avatar with border, we have added a border to the avatar image, as it is adding styling to our profile picture.
We can add a border to both circular as well as square shapes using the class name avatar-image and border-image classes.

Alert
Provide contextual feedback message for typical user actions with the handful of available and flexible alert messages. We have different alert types as specified below.
Primary Alert

In Primary ALert we are using class as msg-alert
IT is a simple and informative message that popped up for the user.

done Thank You

Success Alert

As a success alert, we have background color as green and a success sign which done symbol used.
The color and symbol themselves give the message as it is completed/ successful. We used class msg-alert and success

done Apps Updated

Warning Alert

As a Warning alert, we have a background color as red and a warning symbol used. The color and symbol themselves give the message as is warning us that the action is undone.
We used class as msg-alert and danger.

warning Action can not be Undone

Error Alert

As an error alert, we have a background color as yellow and an error symbol used.
We have used class as msg-alert and error.

Button
Button used for actions in forms, dialogs, and more support for multiple sizes, states and more.
Simple Button

Using a simple Button, the button is used in black and white form with the border as radius 0.2rem which makes it a little curved at the corner of the button.
We use class btn and type submit with it.t

Primary Button: use class primary-btn.

Using Primary Button, the background is colored using the Primary color of our theme of Website. Here we have used purple background color and text color as white.
We use class btn and primary-btn.

Link Button: use class link-btn.

The link button means the link is enclosed in the button tag, which makes it a link Button. Sometimes we need to direct a user to the other page with the button click, then we use the link button.
We use our btn and link-btn class to style it.

Icon with Button: use class icon-btn .

Here we are using simple button with Icon to show the direct message to the user.
We use class btn and icon-btn.

Floating Action Button: use class float-btn.

Floating action means if a user hovers or click on the button it should show some action or click on that button, so we added hover to the button to show float on to the button
We use class btn and float-btn with it.
